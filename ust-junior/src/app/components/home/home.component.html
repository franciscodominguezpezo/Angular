UST Junior Frontend Project

Este proyecto ha sido desarrollado como parte de mi postulación para el puesto de Desarrollador Frontend en UST referencia RH4185706.

La aplicación está construida con Angular Standalone, JavaScript/TypeScript y Bootstrap, siguiendo buenas prácticas de arquitectura frontend, componentes reutilizables y simulación de integración con backend.

El objetivo es demostrar mi capacidad para traducir maquetas a código funcional, construir la capa de presentación y trabajar con componentes interactivos orientados al usuario final.
